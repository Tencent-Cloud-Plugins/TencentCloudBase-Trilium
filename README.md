# [Trilium](https://github.com/zadam/trilium)

Trilium Notes 是个层级结构的笔记应用，专注构建个人的大型知识库。

## 部署

本项目基于开源项目 [CloudBase Framework](https://github.com/Tencent/cloudbase-framework) 开发部署，支持一键云端部署
